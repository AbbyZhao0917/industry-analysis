# 市场规模测算

> 来源：肖璟《如何快速了解一个行业》第3章

## TAM / SAM / SOM 三层模型

```
┌─────────────────────────────────────┐
│ TAM (Total Addressable Market)      │
│ 总可寻址市场                         │
│ "如果全世界都用我们的产品"            │
│  ┌───────────────────────────────┐  │
│  │ SAM (Serviceable Available)    │  │
│  │ 可服务市场                      │  │
│  │ "在我们能力范围内的市场"          │  │
│  │  ┌─────────────────────────┐  │  │
│  │  │ SOM (Serviceable        │  │  │
│  │  │ Obtainable)             │  │  │
│  │  │ 可获取市场               │  │  │
│  │  │ "短期内实际能拿到的"       │  │  │
│  │  └─────────────────────────┘  │  │
│  └───────────────────────────────┘  │
└─────────────────────────────────────┘
```

**使用场景**：
- TAM：向投资人展示市场想象空间
- SAM：中期（3-5年）目标
- SOM：短期（1-2年）可达目标

## 两种测算方法

### 方法一：自上而下（Top-down）

从宏观数据出发，层层拆解：

```
总人口（或总企业数）
  × 目标用户占比（%）
  × 渗透率（%）
  × 年均消费频次（次）
  × 客单价（元）
  = 市场规模（元/年）
```

**示例——中国咖啡市场规模**：
```
14亿人
  × 城市人口60% = 8.4亿
  × 咖啡饮用者占比30% = 2.52亿
  × 年均消费150杯
  × 客单价20元
  = 7560亿元/年
```

**优点**：逻辑清晰，容易沟通
**缺点**：每个假设都可能偏差，容易"垃圾进垃圾出"

### 方法二：自下而上（Bottom-up）

从微观数据加总：

```
门店数 × 单店年均营收
或
用户数 × 单客年均消费（ARPU）
```

**示例——瑞幸咖啡自下而上**：
```
瑞幸门店数 10000家
  × 单店日均出杯量 300杯
  × 年均营业天数 350天
  × 客单价 15元
  = 约157亿元/年
```

**优点**：数据来自实际运营，更可靠
**缺点**：需要较细的微观数据

### 交叉验证

两种方法都做，对比结果：
- 如果结果接近（差异 < 30%）：互相印证，可信度高
- 如果结果差异大：找出假设偏差的原因，修正模型
- **最终取值应该偏保守**：取两种方法的下限

## 渗透率判断法

这是全书最核心的定量框架之一，来自第1章：

| 渗透率 | 阶段 | 核心含义 |
|--------|------|---------|
| < 15% | 导入期 | 市场还在教育用户 |
| 15-35% | 成长期 | 市场快速扩张，增速最高 |
| 35%+ | 成熟期 | 增量空间收窄，存量竞争为主 |

**渗透率 = 存量用户数 / 潜在客户总数**

注意：
- "潜在客户总数"的定义很关键，定义太大会低估渗透率，太小会高估
- 不同行业的潜在客户定义不同（B2C看人口，B2B看企业数）
- 有些行业（如白酒）渗透率很难定义，可结合年龄、收入等维度定义"目标人群"

## 市场规模的边界问题

书中提醒的几个注意事项：

1. **市场边界会变**：跨界竞争让"行业"边界模糊（如美团从外卖切入社区团购）
2. **替代品会扩大/缩小市场**：新能源车替代燃油车，但整个汽车市场可能不变
3. **政策会改变市场边界**：双减政策直接改变了教培行业TAM
4. **技术会创造新市场**：AI Agent在创造全新的企业服务市场
