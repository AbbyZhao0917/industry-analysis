# 竞争格局与盈利性分析

> 来源：肖璟《如何快速了解一个行业》第5章

## 横向格局：行业集中度

### CRn（Concentration Ratio）

前n家企业市场份额之和：

```
CRn = Σ(前n家企业市场份额)
```

| CR4 区间 | 市场类型 | 特征 | 分析重点 |
|----------|---------|------|---------|
| CR4 > 60% | 寡头垄断 | 少数几家主导 | 龙头壁垒、利润率稳定性 |
| CR4 30-60% | 集中市场 | 龙头明确但竞争激烈 | 龙头是否有持续优势 |
| CR4 < 30% | 分散市场 | 无绝对龙头 | 是否有集中度提升趋势？谁在整合？ |

**中国典型行业的CR4参考**：
- 空调：CR4 > 80%（格力、美的、海尔、奥克斯）——高度集中
- 白酒：CR4 约30%（茅台、五粮液、洋河、泸州老窖）——按收入算
- 餐饮：CR4 < 5% ——极度分散
- 快递：CR4 约60%（中通、圆通、韵达、申通）

### 集中度提升的逻辑

分散行业中的投资机会 = **集中度提升趋势**

驱动集中度提升的因素：
1. **规模效应**：大企业成本更低 → 挤掉小企业
2. **品牌化**：消费者从"选品类"变为"选品牌"
3. **政策推动**：环保、安全标准提高淘汰小企业
4. **技术变革**：数字化让大企业管理效率更高
5. **资本推动**：龙头上市后用资本优势并购整合

## 纵向格局：产业链利润分配

### 分析方法

1. 画出产业链（上游→中游→下游→终端）
2. 标注每个环节的代表性企业
3. 对比各环节的毛利率/净利率
4. 找出**最赚钱的环节**和**最不赚钱的环节**
5. 分析原因：为什么这个环节利润高？

### 产业链利润分配示例

**新能源汽车产业链**（简化）：

```
上游矿产（锂、钴、镍）
  → 毛利率 40-60%  ← 资源稀缺，利润丰厚
中游电池（宁德时代等）
  → 毛利率 20-30%
中游零部件
  → 毛利率 15-25%
下游整车（比亚迪、特斯拉等）
  → 毛利率 15-25%
下游充电/服务
  → 毛利率 20-40%
```

### 产业链核心驱动环节

找到产业链中**带动上下游**的关键环节：

| 产业链 | 核心驱动环节 | 逻辑 |
|--------|-------------|------|
| 新能源汽车 | 下游整车销量 | 整车卖得好→电池/零部件/矿产都有需求 |
| 5G | 中游基站铺设 | 基站建好→终端设备和应用才有基础 |
| 半导体 | 下游消费电子+汽车需求 | 终端需求拉动芯片出货 |
| 房地产 | 中游房地产开发 | 上拉建材/工程机械，下推家具/家电 |
| IDC/AI | 中游软件收入增速 | 软件需求拉动算力基础设施 |

## 五力模型

波特五力模型是评估行业盈利能力的经典工具：

| 五力 | 核心问题 | 越强对行业越？ |
|------|---------|--------------|
| **现有竞争者** | 行业内竞争有多激烈？ | 越激烈越不利 |
| **新进入者威胁** | 新人进来有多容易？ | 越容易越不利 |
| **替代品威胁** | 有没有可替代的方案？ | 越多越不利 |
| **供应商议价力** | 上游能不能抬价？ | 越强越不利 |
| **买方议价力** | 下游能不能压价？ | 越强越不利 |

**五力评估结论**：
- 五力都弱 → 行业盈利能力极强（理想投资标的）
- 五力都强 → 行业盈利能力极弱（尽量避开）
- 混合情况 → 看最主要的一两力

## 赔率 vs 概率

书中的一个实用决策框架：

| | 高概率 | 低概率 |
|--|-------|-------|
| **高赔率** | 最佳：龙头还有很大空间 | 风投型：赌黑马 |
| **低赔率** | 稳健型：龙头但增速慢 | 最差：不投 |

- **概率** = 判断正确的可能性（确定性）
- **赔率** = 如果正确能赚多少（回报空间）

在集中度提升中的体现：
- 龙头提升市占率 = **高概率**（趋势明确）
- 分散行业中的搅局者 = **高赔率**（可能爆发）
