# 研究数据源大全

> 来源：肖璟《如何快速了解一个行业》第12章

## 一、统计机构

### 中国官方统计

| 机构 | 数据类型 | 网址 | 更新频率 |
|------|---------|------|---------|
| **国家统计局** | GDP、CPI、PMI、人口、社零、工业增加值、固定资产投资等 | stats.gov.cn | 月/季/年 |
| **中国人民银行** | 利率、货币供应(M2)、信贷、外汇储备 | pbc.gov.cn | 月/季 |
| **中国海关总署** | 进出口数据 | customs.gov.cn | 月 |
| **商务部** | 消费市场、外资、对外投资 | mofcom.gov.cn | 月/季 |
| **工信部** | 工业运行、通信业数据 | miit.gov.cn | 月/季 |

## 四、公司公开信息

| 信息类型 | 获取渠道 | 内容 |
|---------|---------|------|
| **招股书** | 上交所/深交所/港交所/纽交所网站 | 行业分析章节、商业模式、财务数据——最全面的公开信息 |
| **年报/季报** | 同上 | 最新财务数据、经营分析 |
| **投资者演示材料** | 公司投资者关系网站 | 业务亮点、战略方向 |
| **投资者日/业绩会** | 公司网站、东方财富等 | 管理层直接交流 |

**招股书是最重要的免费信息源**：行业分析章节通常委托第三方咨询公司撰写，数据翔实可靠。

## 七、信息检索技巧

书中提到的实用技巧：

1. **搜索格式**：`<关键词> site:<网站>` 限定来源
2. **文件类型搜索**：`<关键词> filetype:pdf` 找研报
3. **时间限定**：搜索工具中限定近1年/近1月
4. **公众号搜索**：微信搜一搜找行业KOL的分析文章
5. **研报标题模式**：搜索"<行业名> 深度报告"或"<行业名> 年度策略"
