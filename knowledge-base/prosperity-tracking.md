# 景气度跟踪框架

> 来源：肖璟《如何快速了解一个行业》第8章

## 景气度的定义

**景气度** = 宏观经济或某一行业的繁荣程度。景气度高 = 营收和利润增速快。

**景气度指标** = 能高频更新、反映行业各环节经营情况的数据指标。两个条件：
1. 能反映行业基本面的业绩情况
2. 更新频次足够高（日/周/月，不是年）

## 两种景气度分析方法

### 方法一：产业链上下游推导

关键在于找到产业链中的**核心驱动环节**，然后向上下游推导：

```
核心驱动环节 → 上游需求 → 下游供给
```

**常见产业链核心驱动环节**：

| 产业链 | 核心驱动环节 | 推导方向 |
|--------|-------------|---------|
| 新能源汽车 | 下游：整车销量 | 整车↑ → 电池↑ → 锂矿↑ |
| 5G | 中游：基站铺设量 | 基站↑ → 设备↑ + 终端↑ |
| IDC/AI | 中游：软件/算力需求 | 需求↑ → 服务器↑ → 芯片↑ |
| 半导体 | 下游：消费电子+汽车 | 终端↑ → 芯片↑ → 晶圆↑ |
| 房地产 | 中游：房地产开发 | 开发↑ → 建材↑ + 家电↑ |

### 方法二：行业关键指标跟踪

按行业类型选择核心指标：

| 行业类型 | 核心景气度指标 | 数据频率 | 数据来源 |
|---------|---------------|---------|---------|
| **上游原材料** | 大宗商品价格、库存水平 | 日/周 | 期货交易所、行业协会 |
| **中游制造业** | 产销量、产能利用率、PMI | 月 | 统计局、行业协会 |
| **必选消费** | CPI、社零数据 | 月 | 国家统计局 |
| **可选消费** | 房地产数据、居民收入预期 | 月 | 统计局、央行 |
| **服务业** | PMI服务业、宏观经济指标 | 月 | 国家统计局 |
| **TMT/新兴行业** | 出货量、渗透率、用户数 | 月/季 | IDC、Gartner、公司财报 |

## 白酒行业景气度案例（书中经典案例）

### 核心指标：批价（批发价）

以散瓶飞天茅台批发价为例：

| 优势 | 说明 |
|------|------|
| 数据可得性强 | 百荣酒价、今日酒价等微信公众号每日更新 |
| 及时反映供需 | 批价上涨→需求旺盛→推动公司业绩 |

**茅台股价与批价相关系数高达 0.95**，五粮液相关系数 0.81。

### 辅助指标

| 指标 | 用途 | 信号 |
|------|------|------|
| 经销商库存 | 判断渠道健康度 | 远超往年水平→警惕库存积压 |
| 经销商打款情况 | 判断渠道预期 | 提前打款→看好后市；推迟打款→谨慎 |
| 开瓶率 | 判断真实消费 vs 囤货 | 高开瓶率→健康；低开瓶率→泡沫 |

## 指标分类：前瞻性 vs 同步

| 类型 | 定义 | 示例 |
|------|------|------|
| **前瞻性指标** | 能感知未来趋势，提前预警 | 订单量（先于收入）、经销商打款（先于出货）、招聘量（先于扩张） |
| **高频同步指标** | 及时跟踪验证当前状态 | 月度销量、周度价格、日度客流 |

## 另类数据（Alternative Data）

书中第12章提到的新兴数据源：

| 另类数据类型 | 可分析什么 | 来源 |
|-------------|-----------|------|
| 卫星图像 | 工厂开工率、港口吞吐量、农田面积 | 商业卫星公司 |
| 招聘数据 | 企业扩张/收缩趋势 | 招聘网站公开数据 |
| 交通流量 | 零售客流、物流活跃度 | 地图API、交通部门 |
| 信用卡/支付数据 | 消费趋势 | 第三方数据商 |
| 电商评论/销量 | 产品热度、消费者反馈 | 电商平台公开数据 |
| 社交媒体情绪 | 品牌口碑、行业热度 | 微博、小红书、抖音 |

## 景气度实操步骤

1. **定位行业类型**：上游/中游/下游/消费/TMT？
2. **识别核心驱动**：产业链中哪个环节驱动全局？
3. **确定2-3个核心指标**：不要太多，关键的要每周/每月盯
4. **建立跟踪表格**：记录历史数据 + 当前值 + 趋势方向
5. **定期回顾**：月度更新判断，季度做深度复查

## 注意事项

1. **不要被短期波动误导**：单月数据波动可能是噪音，看3个月趋势
2. **区分周期性和结构性**：是周期性调整还是结构性下行？
3. **库存是双刃剑**：库存增长可能是补库（看好）也可能是滞销（看空）
4. **交叉验证**：多个指标指向同一方向才可信
