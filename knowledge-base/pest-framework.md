# PEST 外部因素分析框架

> 来源：肖璟《如何快速了解一个行业》第7章

## PEST 四维框架

### P — 政治因素 (Political)

| 检查项 | 说明 | 示例 |
|--------|------|------|
| **产业政策** | 国家/地方对该行业的态度（鼓励/限制/中性） | 新能源车补贴、半导体大基金、教培双减 |
| **监管法规** | 行业准入门槛和运营规范 | 金融牌照、医药审批、食品安全标准 |
| **税收政策** | 行业税率、税收优惠、出口退税 | 高新技术企业15%所得税、跨境电商退税 |
| **贸易政策** | 关税、进出口限制、制裁 | 中美科技脱钩对半导体影响、稀土出口管制 |
| **反垄断** | 平台经济监管、并购审查 | 互联网平台反垄断、数据安全法 |

### E — 经济因素 (Economic)

| 检查项 | 说明 | 数据来源 |
|--------|------|---------|
| **GDP增速** | 整体经济增速直接影响大部分行业 | 国家统计局 |
| **居民收入** | 可支配收入决定消费能力 | 国家统计局 |
| **利率水平** | 影响企业融资成本和消费信贷 | 央行 |
| **通胀(CPI/PPI)** | 影响定价能力和成本 | 国家统计局 |
| **汇率** | 影响进出口企业、海外业务 | 央行/外汇局 |
| **消费信心** | 消费者支出意愿 | 央行储户调查 |

### S — 社会文化因素 (Social)

| 检查项 | 说明 | 示例 |
|--------|------|------|
| **人口结构** | 年龄、性别、城乡分布变化 | 老龄化→医疗健康；少子化→宠物经济 |
| **消费观念变迁** | 消费者价值观的改变 | 从"拥有"到"使用"（共享经济）；健康意识提升（无糖饮料） |
| **生活方式变化** | 日常行为模式的变化 | 居家办公→远程协作工具；独居增多→一人食/小家电 |
| **教育水平** | 人力资本和消费升级基础 | 高学历人群增加→知识付费、高端消费 |
| **文化趋势** | 社会热点、审美、价值观 | 国潮崛起→国产品牌替代；ESG→绿色消费 |

### T — 技术因素 (Technological)

| 检查项 | 说明 | 分析工具 |
|--------|------|---------|
| **关键技术突破** | 对行业有颠覆性影响的技术 | 关注Gartner Hype Cycle |
| **技术成熟度** | 技术从实验室到产业化的阶段 | Gartner曲线：萌芽→膨胀→破裂→爬升→成熟 |
| **数字化程度** | 行业的数字化渗透水平 | 数字化率、线上渗透率 |
| **AI影响** | AI对行业效率、模式、岗位的影响 | 哪些环节可以被AI重构？ |

### 技术创新层级金字塔

书中提出的分析框架，从上到下：

```
         ┌──────────────┐
         │  生态参与者   │ ← 基于平台的应用和服务
         ├──────────────┤
         │  应用程序     │ ← 面向用户的App和软件
         ├──────────────┤
         │  操作系统     │ ← 平台级基础设施
         ├──────────────┤
         │   硬件        │ ← 终端设备和硬件
         ├──────────────┤
         │ 基础设施      │ ← 底层技术标准/协议
         └──────────────┘
```

技术创新的三个作用：
1. 让新兴行业的商业模式变得可行（移动互联网+智能手机→网约车）
2. 实现降本增效，新品类替代旧品类（新能源车替代燃油车）
3. 创造新渠道，为行业扩张提供路径（电商成为主流零售渠道）

## PEST 分析实操步骤

1. **先扫描后深挖**：先过一遍所有维度，标记影响最大的2-3个因素
2. **区分短期和长期**：政策补贴（可能3-5年）vs 人口老龄化（20-30年趋势）
3. **找"导火线"**：哪个外部因素可能是行业爆发的触发点？
4. **关注交汇点**：多种因素叠加处往往是最大的变化（如：AI + 老龄化 + 政策支持 = 智慧养老）
5. **持续跟踪**：外部因素变化最快，至少每季度更新一次

## 多因素综合作用

书中强调：行业发展壮大往往是多种驱动力共同作用的结果，而非单一因素。

**示例——新能源汽车爆发**：
- 政策：补贴、双积分政策、禁售燃油车时间表
- 技术：电池成本下降80%+、续航突破
- 经济：油价波动 + 电动车使用成本优势
- 社会：环保意识提升 + 新能源车成"潮品"
