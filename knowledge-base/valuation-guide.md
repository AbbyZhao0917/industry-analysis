# 估值指南

> 来源：肖璟《如何快速了解一个行业》第6章

## 估值通用公式

```
公司市值 = 净利润 × 市盈率（P/E）
```

市值由两部分驱动：
- **净利润**（基本面）：公司赚钱能力
- **市盈率**（情绪/预期）：市场愿意为每元利润付多少钱

## 不同生命周期阶段的估值方法

| 生命周期 | 特征 | 适用估值方法 | 关键指标 | 注意事项 |
|---------|------|-------------|---------|---------|
| **导入期** | 无利润甚至无收入 | VC估值法、可比公司对标 | 市场空间、团队、技术 | 不确定性极高，估值波动大 |
| **成长期** | 收入高增长、利润可能未释放 | PEG、PS（市销率） | 营收增速、用户增长、毛利率趋势 | 重点看增速能否持续 |
| **成熟期** | 收入利润稳定增长 | PE、PB、EV/EBITDA | 利润稳定性、分红率、ROE | 重点看估值性价比 |
| **衰退期** | 收入利润下滑 | 资产重估、清算价值 | 资产净值、清算价值 | 低PE不一定是"便宜" |

## 主要估值指标详解

### PE（市盈率）
```
PE = 股价 / 每股收益 = 市值 / 净利润
```
- 最常用的估值指标
- 适用于稳定盈利的企业
- PE高 = 市场预期高增长；PE低 = 市场预期低增长或有问题

### PEG（市盈增长比）
```
PEG = PE / 盈利增长率(%)
```
- PEG < 1 = 可能被低估
- PEG = 1 = 合理估值
- PEG > 2 = 可能被高估
- 适用于成长期企业

### PB（市净率）
```
PB = 股价 / 每股净资产 = 市值 / 净资产
```
- 适用于重资产行业（银行、地产、制造业）
- PB < 1 = 跌破净资产（可能被低估，也可能是资产质量差）

### PS（市销率）
```
PS = 市值 / 营收
```
- 适用于有收入但没利润的成长期企业
- 常用于SaaS、互联网公司

### EV/EBITDA
```
EV = 市值 + 总债务 - 现金
EBITDA = 息税折旧摊销前利润
EV/EBITDA = 企业价值 / EBITDA
```
- 剔除资本结构和折旧政策差异
- 适用于重资产、高负债行业

## 估值方法选择决策树

```
有稳定利润吗？
├── 是 → 成熟期？ → PE + PB + EV/EBITDA
└── 否 → 有收入吗？
    ├── 是 → 成长期 → PS + PEG
    └── 否 → 导入期 → VC估值法（TAM反推 + 可比公司对标）
```

## 分部估值法（SOTP）

对于多元化业务公司，按业务板块分别估值再加总：

```
公司总价值 = Σ(各业务板块价值) - 集团费用
```

**适用场景**：
- 综合性集团（如复星、平安）
- 多业务互联网公司（如腾讯=社交+游戏+金融科技+投资）

## 估值的水位判断

除了看绝对估值数字，还需要看相对位置：

1. **历史分位**：当前PE在过去5年处于什么位置？
2. **行业对比**：相对同行是贵还是便宜？
3. **国际对比**：相对海外同类公司？
4. **市场整体**：相对大盘是贵还是便宜？

## 常见估值陷阱

1. **低PE不等于便宜**：可能是利润已到顶峰（周期股高点）
2. **高PE不等于贵**：利润起点低，高增长可以快速消化估值
3. **PB < 1不一定安全**：资产可能有大量坏账或贬值
4. **PS低不一定好**：可能毛利率极低，做再多收入也不赚钱
5. **利润可被操纵**：关注经营现金流 vs 会计利润
