# 便利店行业研究报告

> 分析日期：2026年6月28日  
> 分析方法：肖璟《如何快速了解一个行业》七维度框架

## 一、行业概览与生命周期判断

**行业定义**：便利店是以即时性消费需求为核心，经营面积通常在 50-200㎡，营业时间 16-24 小时，提供食品饮料、日用百货、鲜食等商品的小型零售业态。

**生命周期判断**：
- **渗透率**：中国便利店覆盖约 32 万门店，按 14 亿人口、人均覆盖密度推算，便利店渗透率约 40-50%（一线城市 60%+，下沉市场 20-30%）
- **阶段**：**成熟期早期**
- **依据**：
  1. 行业增速从两位数放缓至个位数（2025 年销售额增速 8.7%）
  2. 门店净增从 7.7% 回落至 5.6%，闭店量达三年最高（8,539 家）
  3. 行业从"粗放扩张"转向"提质增效"，并购整合加速

> 数据来源：CCFA & 毕马威《2025 中国便利店发展报告》、华经产业研究院

## 二、可行性分析

### 2.1 商业模式画布

| 画布模块 | 内容 |
|---------|------|
| **客户细分** | 城市居民（18-45岁为主），追求便利、时效的消费者；社区住户 + 办公白领 + 夜间消费者 |
| **价值主张** | "随时随地、即时可得"——解决"急需/临时/小量"购买需求 |
| **渠道通路** | 线下门店（核心）+ 即时零售平台（美团/饿了么，渗透率 38%）+ 前置仓 |
| **客户关系** | 会员体系（会员客单价增速 2.0% vs 非会员 0.3%）、高频触达（日用品消费） |
| **收入来源** | 商品销售（烟酒 30%+，鲜食 20-30%↑，日用品）+ 服务费（快递代收、缴费等） |
| **核心资源** | 门店网络密度、供应链效率、品牌认知、选址能力 |
| **关键业务** | 商品采购与选品、门店运营、供应链管理、即时零售履约 |
| **重要伙伴** | 品牌供应商、外卖平台、加盟商、物流配送 |
| **成本结构** | 商品成本（~73.6%）、租金（~8-10%）、人工（~8-10%）、物流与其他 |

### 2.2 对标验证

- **时间对标**：人类对"即时可得"的需求自古存在——从村口小卖部→供销社→夫妻杂货店→连锁便利店→即时零售。需求不变，满足方式在持续升级。
- **空间对标**：日本便利店人均覆盖约 2,200 人/店，中国约 4,300 人/店（按 32 万店计算），密度仍有提升空间。7-11 日本模式（鲜食驱动 + 高频复购）正在被中国品牌学习。

### 2.3 需求质量评估

- **频次**：高（日用品/食品/饮料，人均周消费 2-3 次）
- **弹性**：小（"急需"场景下几乎不议价，"便利"本身就是付费理由）
- **矩阵定位**：⭐ 频次高 + 弹性小 → **最优需求象限**

## 三、规模性分析

### 3.1 TAM/SAM/SOM

| 层级 | 估算 | 逻辑 |
|------|------|------|
| **TAM** | ~8,000-10,000 亿元 | 全口径零售中的即时消费需求 |
| **SAM** | ~5,033 亿元（2025年销售额） | 目前连锁便利店可服务的市场 |
| **SOM** | 头部企业有较大提升空间 | 美宜佳 19.3% 市占率，CR5 约 50% |

### 3.2 自上而下测算

```
14亿人口
  × 城镇人口 67% = 9.38亿
  × 便利店触达率 80% = 7.5亿
  × 人均周消费 1.5次 × 52周 = 78次/年
  × 客单价 16.5元
  ≈ 9,653 亿元（TAM级别）
```

### 3.3 增长驱动力

- **下沉市场渗透**：三四线城市和县域便利店密度远低于一线城市
- **鲜食占比提升**：从 20% 向日本 35-40% 靠拢，拉高客单价和毛利率
- **即时零售**：线上渗透率从 38% 继续提升，扩大覆盖半径
- **夜间经济**：24 小时门店从 5.3 万家继续增长

## 四、防守性分析

### 4.1 护城河评分卡

| 护城河子项 | 评分 | 评语 |
|-----------|------|------|
| 资金/资本门槛 | 2/5 | 单店投入 30-80 万，门槛不高 |
| 技术/专利壁垒 | 2/5 | 缺乏核心技术壁垒 |
| 牌照/准入限制 | 2/5 | 烟草证有一定门槛但总体不高 |
| 数据独占性 | 3/5 | 会员消费数据有累积优势 |
| 自然资源控制 | 1/5 | 不适用 |
| 网络效应 | 3/5 | 门店密度带来便利性网络效应（密度越高越方便） |
| 规模效应 | 4/5 | 采购规模大→进价低→利润空间大 |
| 转换成本 | 2/5 | 消费者换店成本极低 |
| 品牌/渠道 | 3/5 | 美宜佳/罗森/7-11有品牌认知，但价格敏感型消费者多 |
| **总分** | **22/45** | 🚧 **一般护城河** |

### 4.2 护城河判断

便利店行业的护城河整体偏弱。最强的护城河来自**规模效应**（采购成本优势）和**门店密度网络效应**（越密越方便）。但消费者转换成本极低——隔壁开一家更便宜的就可能流失。这也是行业净利率仅 2.5% 的深层原因。

## 五、盈利性分析

### 5.1 行业集中度

| 指标 | 数值 | 判断 |
|------|------|------|
| CR3 | ~42.6%（美宜佳+易捷+昆仑好客） | 中等集中 |
| CR5 | ~50% | 中等集中，但较 2023 年（~30%）大幅提升 |
| CR10 | ~58% | |
| **趋势** | 集中度快速提升中 | 头部扩张 + 尾部闭店加速 |

### 5.2 产业链利润分配

```
上游供应商（品牌商）
  → 毛利率 30-50%（品牌商品）
中游便利店
  → 毛利率 26.4%，净利率仅 2.5%
    ← 利润极薄，赚的是"辛苦钱"
下游消费者
  → 为"便利"付出约 10-20% 的价格溢价
平台方（外卖平台）
  → 抽佣 15-22%
```

**结论**：便利店自身赚的是产业链中最薄的钱。品牌商和平台方利润更厚。

### 5.3 五力模型

| 五力 | 强度 | 说明 |
|------|------|------|
| 现有竞争者 | ⭐⭐⭐⭐⭐ 极强 | 密度高、差异化低、价格战频繁 |
| 新进入者威胁 | ⭐⭐⭐ 中等 | 单店门槛低但连锁化有壁垒 |
| 替代品威胁 | ⭐⭐⭐⭐ 强 | 即时零售/社区团购/无人零售/前置仓都在抢食 |
| 供应商议价力 | ⭐⭐⭐ 中等 | 品牌商强势（烟酒/饮料），自有品牌可降依赖 |
| 买方议价力 | ⭐⭐⭐⭐⭐ 极强 | 消费者换店零成本 |

**综合**：便利店行业五力都偏强，是典型的**辛苦生意**。

## 六、估值分析

### 6.1 适用估值方法

行业处于**成熟期早期**，适用：
- **PE 估值**：有稳定利润，但增速放缓
- **PS 估值辅助**：利润率薄，PS 也有参考意义

### 6.2 估值参考

| 指标 | 数值 |
|------|------|
| 行业平均毛利率 | 26.4% |
| 行业平均净利率 | 2.5% |
| 可比估值 | 日本 7-11（7&i Holdings）PE 约 15-20x，全家/罗森 PE 约 15-25x |

便利店不是高估值行业——薄利、辛苦、竞争激烈。估值溢价来自**规模壁垒**和**品牌效应**，只有头部企业能享有。

## 七、外部因素分析（PEST）

### 7.1 政治 (P)
- ✅ 商务部推动"一刻钟便民生活圈"建设，利好社区便利店
- ✅ 各地放宽外摆和夜间经营限制
- ⚠️ 烟草监管趋严（烟草占便利店收入 30%+，是核心引流品类）

### 7.2 经济 (E)
- ⚠️ 消费降级趋势下，消费者对价格更敏感，客单价承压（2024 年单店日营收降至 4,634 元）
- ✅ 下沉市场消费升级，县域便利店增长空间大
- ⚠️ 租金和人工成本持续上升，挤压利润

### 7.3 社会文化 (S)
- ✅ 独居人口增加 + 小家庭化 → "少量多次"购买需求增加
- ✅ 夜生活丰富化 → 24 小时消费需求上升
- ✅ 健康意识提升 → 鲜食/轻食/健康零食需求增长

### 7.4 技术 (T)
- ✅ **AI 智能补货**：库存周转天数下降 6.7%，报损减少 14.6%
- ✅ **云值守**：夜间运营成本降低 65-70%
- ✅ **即时零售**：38% 便利店已接入，扩大 3km 覆盖半径
- ✅ **数字化会员**：会员客单价年均增长 2.0%

### 8.1 关键跟踪指标

| 指标类型 | 核心指标 | 当前数值（2024-2025） | 趋势 |
|---------|---------|---------------------|------|
| 营收端 | 单店日均营收 | 4,634 元 | ↓ 微降 |

### 8.2 当前景气度判断

**中性偏冷**。销售额仍增长（+8.7%），但单店日营收下滑、闭店量攀升、净利率仅 2.5%，说明行业整体"增收不增利"，景气度承压。

## 九、综合评估

### 9.1 行业吸引力评分：**6/10**

| 维度 | 得分 | 理由 |
|------|------|------|
| 可行性 | 8/10 | 需求真实且高频 |
| 规模性 | 7/10 | 5000亿+，还在增长 |
| 防守性 | 4/10 | 护城河弱，22/45分 |
| 盈利性 | 3/10 | 净利率仅2.5%，极薄 |
| 估值 | 5/10 | 不算贵但溢价空间有限 |
| 外部因素 | 6/10 | 政策利好但消费降级承压 |
| 景气度 | 5/10 | 中性偏冷 |

### 9.2 核心风险

1. **利润过薄**：2.5% 净利率，任何成本波动都可能吞噬利润
2. **替代品侵蚀**：即时零售前置仓利润率 3-10% 高于传统门店 1-5%
3. **闭店潮**：2025 年闭店 8,539 家达三年最高
4. **烟草监管**：若烟草销售受限，将重创便利店核心引流品类

### 9.3 投资/进入建议

- **适合谁**：有供应链优势、能承受薄利、想做规模生意的企业
- **不适合谁**：追求高利润率、快速回本的投资者
- **关键成功因素**：规模（够大才能压采购成本）+ 鲜食能力（提升毛利）+ 数字化（降本增效）
- **一句话**：便利店是一门"辛苦但真实"的生意——需求永远存在，但赚钱不容易。只有做到头部的规模玩家才有可观的回报。
